## Overview

Simple Bluetooth board designed with STM32WB55.

## Features

- Bluetooth
- USB Serial
- 2 x Buttons
- 2 x Indicator LEDs
- 1 x Analog Switch

## Main Components

- Microcontroller: STM32WB55CCU6
- Sensors/Modules: 
- Power Requirements: 5V, xmA

## Software Requirements

- Altium Designer 24.3.1
